Predicting Boston Housing Prices

Built a model to predict the value of a given house in the Boston real estate market using various statistical analysis tools. Identified the best price that a client can sell their house utilizing machine learning. Technologies: Python, bias-variance, cross validation, learning curves, statistical analysis, IPython notebook

Train a smartcab in a simulated environment

Applied reinforcement learning techniques for a self-driving agent in a simplified world to aid it in effectively reaching its destinations in the allotted time. Implemented a Q-Learning algorithm for the self-driving agent to guide the agent towards its destination within the allotted time; also found the best configuration of learning and exploration factors to ensure the self-driving agent is reaching its destinations with consistently positive results. Technologies: Python, reinforcement learning (Q-learn), IPython notebook

Finding Donors for CharityML

Investigated factors that affect the likelihood of charity donations being made based on real census data. Developed a naive classifier to compare testing results to. Trained and tested several supervised machine learning models on preprocessed census data to predict the likelihood of donations. Selected the best model based on accuracy, a modified F-scoring metric, and algorithm efficiency. Technologies: Python, sklearn, Decision Trees, support vector machines, k-nearest neighbors, model fitting and prediction, IPython notebook
