Machined Heuristics by Manish

Exploring Survival on Titanic

Posed a question about the Titanic passenger dataset, then used NumPy and Pandas to answer that question based on the data and created a report to share the results.

Using Machine Learning to Identify Fraud

Utilizing machine-learning tools to build an algorithm to identify Enron employees who may have committed fraud based on the public Enron financial and email dataset. Explored five different algorithms (GaussianNB, Support Vector Machines, Decision Tree, K Neighbors, and Logit)

Wrangle OpenStreetMap Data

Chose a region (Raleigh-Durham, NC) and used data munging techniques to assess the quality of the data for validity, accuracy, completeness, consistency and uniformity.

Designing an A/B Test

Designed an A/B test, including which metrics to measure and how long the test should be run. I also analyzed the results of an A/B test that was run by Udacity, recommended a decision, and proposed a follow-up experiment.

Data Exploration using R

Investigated the Prosper Loan dataset using R and exploratory data analysis techniques, exploring both single variables and relationships between variables.

Applied Machine Learning

Implemented a K-means clustering algorithm and applied the algorithm to compress an image; used Principal Component Analysis to find low dimensional representation of face images Developed an anomaly detection algorithm in server computers to detect any harmful behavior using Gaussian distribution Technologies used: Matlab/Octave
