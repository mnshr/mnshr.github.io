MHAD Data Challenge¶

Introduction¶

Hcange Nifancial is a regional bank based in Washington, DC. The leadership in Hcange Nifancial wants to explore entering a new market, home lending in the surrounding states such as MD, VA, WV, DL.

In this study I have analyzed the MHAD (Home Mortgage Disclosure Act) data set, published by the federal government every year. The study tries to answer questions about the regional home loans market.

In this section I get the data loaded from the csv files and merge the home loans application data with the institutional loan data based on 'Respondent_Name'. I also create an additional column 'Loan_Groups' to categorize loans based on 'Loan_Amount_000'.

# Function to categorize loans based on loan amounts
def mhad_loan_groups(ln_amt):
    """
    Create 6 groups of Loan_Amount_000
    """
    if ln_amt < 100:
        return '0-100'
    elif ln_amt < 200:
        return '100-200'
    elif ln_amt < 400:
        return '200-400'
    elif ln_amt < 800:
        return '400-800'
    elif ln_amt < 1000:
        return '800-1000'
    else:
        return '1000-'

# Add the new bucket
merged_data['Loan_Group'] = merged_data.Loan_Amount_000.apply(mhad_loan_groups)

The code block below shows the 2 API functions mhad_init() and mhad_json():

def mhad_init(loan_file, inst_file):
    """
    Reads the data files and mergees the loan data with instituional data; so
    that each loan application is assigned a "Respondent_Name" from institutional
    data. And returns an object containing the expanded MHAD df
    """
    loan_data = pd.read_csv(loan_file)
    inst_data = pd.read_csv(inst_file)
    merged_data = pd.merge(loan_data, inst_data,
                           left_on=['As_of_Year','Agency_Code','Respondent_ID'],
                        right_on=['As_of_Year','Agency_Code','Respondent_ID'],
                        how='left')

    # Add the new bucket
    merged_data['Loan_Group'] = merged_data.Loan_Amount_000.apply(mhad_loan_groups)
    return merged_data

def mhad_to_json(data, states=None, conventional_conforming=None):
    """
    Exports the merged data set to disk, for states filtered by product segment
    The file is created by the name mhad_data.json in the same directory
    """
    if not states:
        states = data['State'].unique()

    if not conventional_conforming:
        conventional_conforming = ['Y', 'N']

    json_data=data[data.State.isin(states) &
    data.Conventional_Conforming_Flag.isin(conventional_conforming)]

    json_data.to_json('./mhad_data.json',orient='index')
    return

Data Quality Summary¶

As a set of rules for monitoring data quality, I would recommend the following set of rules to be kept in mind: A standard set of checks must be performed on every critical field/variable, which include:

* Check for Nulls and drop them
* Check for NaN values and see if they can be replaced with a reasonable alternative
* Have a defined datatype for each field, and if not then try to convert them to assist effective analysis
* Look for abonormal values in a set of data elements which deviate from the type of the data field
* Look for outliers and if they can and should be removed or remain a part of the data analysis
* Plot the fields to understand distribution among themselves in a univariate chart or in relation with others in a bivariate chart

Summary¶

There are multipe things to be observed from the graphs above. One clear industry trend is that volumes are on decline, which is not very encouraging. Apart from this, the mean loan amount is also seen on decline path over the last 3 years. Hence, at the very first look, the market looks to be shrinking.

When we look at the loan type, between purchase and refinance; the refinance market seems much bigger, but has seen a big decline. The purchase market has remained steady on the other hand. It seems that the refinance market has been overstaturated with assets changing hands to squeeze out the last % basis points of savings due to heavy competition. It also seems that lenders are comfortable refinancing an already financed loan as the initial cost incurring due diligence would have been completed and refinance wouln't require as much due diligence. It would be really interesting to see the markets just for the 1st time bought inventory as well as the market of 1st time home buyers. Between refinance and purchase, the asset value is only slightly less for refinance than for purchase. I think with property prices holding and some of the loan paid off over some time, refinancing would require a lesser loan amount.

Applicant incomes show slight decline over the three years but relatively lesser decline than that of the loan amounts. The applicant income is proportional to the loan group for which they apply. The 200k-400k USD group is most popular in terms of number of loans, but has seen a decline over last 3 years. Virginia seems to be the biggest market in terms of volume followed by Maryland.

Based on the observations above I recommend that Hcange Nifancial should not be considering an entry in the home loan market till the trends reverse. If they still wanted to expand their product offering keeping in mind a longer term goal, they should consider an acquisition of an existing lender of suitable size in a specific part of the region. But, determining an acquisition target will be another exhaustive exercise. An acquisition can give them a quick entry in the market where they can test the waters before deciding to expand further or exit.

# This function will be called when the button on interactive display is clicked
def on_button_clicked(b):
    clear_output()
    print ("Displaying the competition graph for year: ", int(year.value), ", in state: ", str(state.value))
    
    resp_cnt = pd.DataFrame({'Mortgage_Count': merged_data.groupby(['As_of_Year', 'Respondent_Name_TS',
                                                                  'State']).size()}).reset_index()
    resp_state = resp_cnt.loc[resp_cnt['State'] == str(state.value)]
    resp_state = resp_state.sort_values(['As_of_Year', 'Mortgage_Count'], ascending=[False, False])
    
    # Get the count of competitors from the slider selection
    num_compt_v=int(num_compt.value)
    # get the top selected lenders for the most recent year
    top_resp = resp_state['Respondent_Name_TS'][:num_compt_v]
    
    # get the data for top institutions
    top_inst_data = resp_state[resp_state['Respondent_Name_TS'].isin(top_resp)].reset_index()
    
    # rearrange the data using pivot for the bar chart
    top_inst_g = top_inst_data.pivot(index='Respondent_Name_TS', columns='As_of_Year',
                                               values='Mortgage_Count').reset_index()
    # size the figure
    rcParams['figure.figsize'] = 12, 7 
    
    # data for selected year
    data1 = np.array(top_inst_g[int(year.value)])
    
    fig, ax = plt.subplots()
    bar_locations = np.arange(num_compt_v)
    ax.bar(bar_locations, data1, label=str(year.value))
    plt.xlabel(top_inst_g['Respondent_Name_TS'])
    plt.title('For state %s | Number of loans from top institutions' % str(state.value))
    fig.subplots_adjust(bottom=0.28)
    plt.xticks(range(10), range(10))

button.on_click(on_button_clicked)
